2. 배열이 0부터 시작하는 이유?

   - arr[0] == *arr(0번 adress+0)
   - arr[1] == *arr(0번 adress+1)
   - 메모리 낭비하지 않기 위해서, 1번부터하면 0번지를 버리는 셈이다.
   - arr[0] == *(arr + 0) == 0[arr]
   - arr[1] == *(arr + 1) == 1[arr]
