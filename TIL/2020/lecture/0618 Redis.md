# 오늘의 강의

### 개요

- Redis
  - 매우 빠른 인메모리 DB
- 하둡의 장점
  - 대용량과 분산처리
- 간단 Key-Value DB로 사용가능
- 주로 캐시로 많이 사용
- 간단 튜토리얼
  - http://try.redis.io/
- Shard
  - 데이터를 저장할 때 나누어 저장하는 방식
- ElasticCache
  - 관리를 AWS에서 해준다
  - 그래서 비싸다
